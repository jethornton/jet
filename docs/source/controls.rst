Controls
========

Use the Name from the list below for each control widget objectName.
::

	Control  Name
	E-Stop   estop_pb
	Power    power_pb
	Run      run_pb
	Step     step_pb
	Pause    pause_pb
	Resume   resume_pb
	Stop     stop_pb

.. note:: You don't have to use all the controls, the ones found will be
  connected to the correct code
